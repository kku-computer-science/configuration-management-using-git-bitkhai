# Sorting Algorithms Project

โปรเจกต์นี้เป็นส่วนหนึ่งของรายวิชา Software Engineering  โดยมีวัตถุประสงค์เพื่อศึกษาและเปรียบเทียบการทำงานของขั้นตอนวิธีในการเรียงลำดับข้อมูลสองรูปแบบ ได้แก่ **Quick Sort** และ **Bubble Sort** พัฒนาด้วยภาษา Python

## รายละเอียดโปรเจกต์
1. ป้อนชุดตัวเลขจำนวนเต็มที่ต้องการเรียงลำดับ
2. เลือกอัลกอริทึมที่ต้องการใช้งาน (Quick Sort หรือ Bubble Sort)
3. รับผลลัพธ์เป็นรายการตัวเลขที่เรียงลำดับแล้ว

## โครงสร้างไฟล์

- `main.py`: ไฟล์หลักสำหรับรันโปรแกรม ทำหน้าที่รับค่าจากผู้ใช้ (Input), เรียกฟังก์ชันการทำงาน และแสดงผลลัพธ์ (I/O Handler)
- `quicksort.py`: โมดูลสำหรับประมวลผลอัลกอริทึม Quick Sort
- `bubblesort.py`: โมดูลสำหรับประมวลผลอัลกอริทึม Bubble Sort
